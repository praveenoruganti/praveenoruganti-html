


# Heading Tags





HTML headings are defined with <h1> to <h6> tags.

The <h1> defines the most important heading whereas <h6> defines the least important heading.

```html
  <h1>First Heading</h1>
  <h2>Second Headng</h2>
  <h3>Third Heading</h3>
  <h4>Fourth Heading</h4>
  <h5>Fifth Heading</h5>
  <h6>Sixth Heading</h6>
```
